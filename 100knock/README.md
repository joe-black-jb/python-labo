# About

- 「Python 実践データ分析 100 本ノック 第 2 版」の学習記録
- 第 8 章までの内容は 100knock/main.py に記載
- 第 9 章からはディレクトリごとに main.py を設置

# 起動方法

```sh
# 第8章まで
[100knock]
python main.py

# 第9章以降
[該当する章のディレクトリ]
python main.py
```
